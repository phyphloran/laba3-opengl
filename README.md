# ЛР OpenGL — Вариант №2

Реализована сцена с произвольной пирамидой и кубом, у которого одно ребро лежит на выбранном ребре пирамиды.

## Управление

- **ЛКМ / ПКМ** — поворот куба вокруг выбранного ребра пирамиды.
- **↑ / ↓** — перемещение куба вдоль выбранного ребра.
- **Цифры `1..9` и `0`** — выбор ребра пирамиды (`0` = ребро №10).

В сцене используется пятиугольная пирамида (10 рёбер: 5 в основании и 5 боковых), поэтому переключение по цифровым клавишам покрывает все рёбра.
